iMuse
=====

一个在线通信聊天系统，界面仿造WeChat，7.23 - 7.27建设中.....
CSS现在只写了布局，没有设计，到时候用bootstrap.css套用
#1.聊天系统设计			
> <img src="./doc/frame.png" alt="设计框图" />

```
 + 前端使用AngularJs来实现数据绑定，实现UI高可用
 + 后台使用ExpressJs构建Web服务
 + Socket.io 事件响应处理方式传播消息
 + Mogoose存储数据
```
